Projeto de Comandos e Anotações

Este repositório reúne meus estudos, práticas e comandos utilizados durante atividades no Kali Linux.
O objetivo é manter tudo organizado e acessível, tanto para consulta futura quanto para compartilhar conhecimento com quem também está aprendendo.

---

➤ Estrutura do Repositório

* **/comandos/**
  Contém arquivos `.txt` com os comandos que utilizei durante as práticas.
  Cada arquivo representa um conjunto de comandos relacionado a um tópico ou ferramenta.

* **/prints/**
  Pasta contendo capturas de tela feitas durante a execução dos comandos e testes.
  Útil para visualizar o comportamento real das ferramentas.

* **README.md**
  Este arquivo que você está lendo.

---

➤ Como navegar pelos arquivos

➤ Comandos

Para visualizar os comandos executados, acesse a pasta:

```
/comandos/
```

Cada arquivo `.txt` contém:

1- O comando utilizado
2- Resultado do comando

➤ Prints

As evidências visuais estão na pasta:

```
/prints/
```

Você pode abrir cada imagem para acompanhar o resultado dos comandos descritos nos arquivos `.txt`.
